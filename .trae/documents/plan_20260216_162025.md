## 项目实施计划

### 1. 项目初始化
- 创建 Next.js 14 项目（App Router）
- 安装依赖：`tailwindcss`, `framer-motion`, `katex`, `lucide-react`

### 2. 核心文件结构
```
openlearner/
├── app/
│   ├── layout.tsx          # 全局布局
│   ├── page.tsx            # 主页面
│   └── globals.css         # 全局样式 + KaTeX
├── components/
│   ├── Header.tsx          # 顶部状态栏（能量值、连胜）
│   ├── LevelMap.tsx        # 蛇形关卡地图
│   ├── LevelNode.tsx       # 3D 质感关卡节点
│   ├── KnowledgeCard.tsx   # 知识点卡片
│   ├── QuizPanel.tsx       # 交互练习面板
│   ├── FeedbackModal.tsx   # 正确/错误反馈弹窗
│   ├── AIAssistant.tsx     # AI 助手侧边栏
│   ├── PDFUploader.tsx     # PDF 上传区域
│   └── ProgressBar.tsx     # 进度条组件
├── data/
│   └── mockLevels.ts       # 3个预设关卡数据
├── lib/
│   └── imagePrompt.ts      # generateImagePrompt 函数
└── types/
    └── index.ts            # TypeScript 类型定义
```

### 3. 视觉实现
- 浅色模式配色方案（#F9FAFB 背景、#FFFFFF 卡片）
- 紫色 #8B5CF6 品牌色 + 亮青 #00C3E3 成功色
- 渐变按钮 `linear-gradient(to right, #8B5CF6, #FBBF24)`
- 卡片圆角 24px + 柔和投影

### 4. 核心功能
- PDF 上传模拟（加载动画 → 生成学习路径 JSON）
- KaTeX 数学公式渲染
- 多选题/排序题交互
- Framer Motion 动效（卡片滑入、按钮交互）
- AI 助手 Mock 对话

### 5. 预设关卡内容
1. **概念引入**：关系的定义与集合表示
2. **性质判断**：自反性、对称性可视化判定
3. **高级挑战**：等价类与商集交互式划分
